# CA DOM atsiskaitymas

## Reikiamos programos
- Node.js
- Naršyklė

## Instaliavimas
- šio failo direktorijoje, įvykdykite komandą:
  - npm install

## Projekto paleidimas
  1. šio failo direktorijoje paleiskite serverį:
    * windows - npm run start-win
    * Unix/Max - npm run start-unix
  2. Naršyklėje atidarykite failą index.html
